# DCF估值参数自定义指南

## 概述

DCF（Discounted Cash Flow，折现现金流）估值模型包含多个可配置参数。这些参数现在可以通过API进行自定义，以适应不同公司的特点和投资者的预期。

## 可配置参数

### 1. 基础参数

| 参数 | 类型 | 默认值 | 说明 | 取值范围 |
|------|------|--------|------|----------|
| `forecast_years` | int | 5 | 预测年数 | 1-10 |
| `terminal_growth` | float | 0.02 | 终值增长率（永续增长） | 0-0.1 |
| `risk_free_rate` | float | 0.03 | 无风险利率 | 0-0.2 |
| `market_return` | float | 0.08 | 市场回报率 | 0-0.3 |

### 2. 税务与成本参数

| 参数 | 类型 | 默认值 | 说明 | 取值范围 |
|------|------|--------|------|----------|
| `tax_rate` | float | 0.25 | 企业所得税率 | 0-0.5 |
| `credit_spread` | float | 0.02 | 债务信用利差 | 0-0.1 |

### 3. 增长率参数

| 参数 | 类型 | 默认值 | 说明 | 取值范围 |
|------|------|--------|------|----------|
| `growth_rate_cap` | float | 0.08 | 收入增长率上限 | 0-0.5 |

### 4. WACC参数

| 参数 | 类型 | 默认值 | 说明 | 取值范围 |
|------|------|--------|------|----------|
| `wacc_min` | float | 0.05 | WACC下限 | 0-0.15 |
| `wacc_max` | float | 0.20 | WACC上限 | 0.05-0.5 |

### 5. Beta参数

| 参数 | 类型 | 默认值 | 说明 | 取值范围 |
|------|------|--------|------|----------|
| `beta` | float | null | Beta系数（不指定则根据行业自动计算） | 0-3 |

## API使用示例

### 默认参数估值

```json
POST /api/valuation/batch
{
  "symbols": ["600519"],
  "methods": "dcf"
}
```

### 自定义参数估值

```json
POST /api/valuation/batch
{
  "symbols": ["600519"],
  "methods": "dcf",
  "dcf_config": {
    "forecast_years": 10,
    "terminal_growth": 0.015,
    "risk_free_rate": 0.035,
    "market_return": 0.09,
    "tax_rate": 0.25,
    "credit_spread": 0.025,
    "growth_rate_cap": 0.05,
    "wacc_min": 0.06,
    "wacc_max": 0.15,
    "beta": 1.2
  }
}
```

## 参数影响分析

### 茅台（600519）示例

| 配置 | 公平价值 | 上涨空间 | WACC | 说明 |
|------|----------|----------|------|------|
| 默认参数 | ¥9,455.53 | 524% | 7.23% | 使用默认配置 |
| 保守参数 | ¥7,244.34 | 378% | 8.12% | 更高的折现率，更低的增长率 |
| 自定义Beta=1.2 | ¥8,147.49 | 438% | 7.88% | 用户指定Beta系数 |

### 参数敏感性

1. **终值增长率（terminal_growth）**
   - 影响最大，因为终值通常占企业价值的70-80%
   - 从2%降至1.5%，估值显著下降

2. **WACC（折现率）**
   - 通过无风险利率、市场回报率和Beta计算
   - WACC越高，估值越低

3. **增长率上限（growth_rate_cap）**
   - 限制收入增长的乐观程度
   - 对于成熟公司，建议使用较低值（5-8%）

4. **Beta系数**
   - 衡量股票相对于市场的波动性
   - 食品饮料行业Beta约为0.9
   - 可根据公司具体情况调整

## 参数设置建议

### 成熟公司（如茅台、银行）

```json
{
  "terminal_growth": 0.015-0.025,
  "growth_rate_cap": 0.05-0.08,
  "beta": 0.8-1.0
}
```

### 成长公司（如科技股）

```json
{
  "terminal_growth": 0.025-0.04,
  "growth_rate_cap": 0.15-0.30,
  "beta": 1.0-1.5
}
```

### 周期性公司（如房地产、原材料）

```json
{
  "terminal_growth": 0.01-0.02,
  "growth_rate_cap": 0.03-0.06,
  "beta": 1.2-1.5,
  "wacc_min": 0.08,
  "wacc_max": 0.15
}
```

### 高风险公司（如小盘股、概念股）

```json
{
  "terminal_growth": 0.01-0.02,
  "growth_rate_cap": 0.10-0.20,
  "beta": 1.5-2.5,
  "credit_spread": 0.03-0.05,
  "risk_free_rate": 0.04-0.05
}
```

## 注意事项

1. **数据完整性**：DCF模型需要完整的财务数据。如果数据缺失，会返回错误信息。

2. **参数合理性**：
   - WACC必须大于终值增长率
   - Beta建议在0.5-2.0之间
   - 增长率应考虑公司历史和行业前景

3. **市场环境**：
   - 无风险利率应根据当前国债收益率调整
   - 市场回报率应根据股票市场预期回报调整

4. **参数验证**：
   - 系统会自动验证参数在合理范围内
   - 不合理的参数会导致估值失败

## 常见问题

### Q: 估值结果为什么远高于/低于市场价格？

A: DCF估值反映的是内在价值，市场价格可能受情绪、资金等因素影响。如果差异很大：
- 检查输入参数是否合理
- 尝试调整增长率假设
- 考虑使用相对估值方法对比

### Q: 如何选择合适的Beta系数？

A: Beta可以通过以下方式确定：
1. 使用行业默认值（系统自动计算）
2. 使用公司历史Beta（如果有数据）
3. 根据公司特征调整（如负债率高则Beta高）

### Q: 终值增长率应该设多少？

A: 建议：
- 成熟行业：1.5-2.5%（接近GDP增长率）
- 成长行业：2.5-4%（考虑行业增长空间）
- 避免使用超过5%的永续增长率

## 策略建议规则

### 概率阈值

> **只有概率 ≥75% 的事件才值得参考**
> - 冲高概率 ≥75%：可作为卖出参考
> - 下探概率 ≥75%：可作为买入参考
> - 概率 <75%：不建议作为操作依据

### T+0 操作仓位

| 日内振幅 | 建议操作仓位 |
|---------|-------------|
| ≥ 5% | 50% |
| 3%~5% | 30% |
| < 3% | 不建议操作 |

### 高开日T+0策略（先卖后买）

| 条件 | 建议 |
|------|------|
| 冲高概率 ≥ 75% 且 下探概率 ≥ 75% | 冲高至P50卖出持仓，下探至P50买回 |
| 冲高概率 ≥ 75% 但 下探概率 < 75% | 冲高至P50卖出持仓，择机买回 |
| 冲高概率 < 75% | 不建议操作 |

### 低开日T+0策略（先买后卖）

| 条件 | 建议 |
|------|------|
| 下探概率 ≥ 75% 且 冲高概率 ≥ 75% | 下探至P50买入，冲高至P50卖出持仓 |
| 下探概率 ≥ 75% 但 冲高概率 < 75% | 下探至P50买入，择机卖出 |
| 下探概率 < 75% | 不建议操作 |
